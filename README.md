## ✏️ Siteyi Güncelleme

Kolay yol — GitHub arayüzü:

1. index.html dosyasına tıkla
2. Sağ üstteki kalem simgesine tıkla
3. Değişiklikleri yap
4. Commit changes butonuna tıkla
5. Site 1-2 dakika içinde güncellenir

Alternatif — Yeni dosya yükle:

1. Bilgisayarda yeni index.html hazırla
2. Add file → Upload files tıkla
3. Dosyayı yükle ve Commit changes yap
